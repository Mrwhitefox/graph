The goal of this project is to optimize the layout of graphs by a physic simulation.

It is coded in C and uses the CSFML (a graphic librairy).
You need to install it to run compile the program.
To install it on Linux, enter in a console:
sudo apt-get install libcsfml-dev libgl1-mesa-dev libglu1-mesa-dev

To compile and launch, see the commentaries in simu.c
